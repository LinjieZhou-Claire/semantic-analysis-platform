1.3 TF-IDF vs Count (One-hot) 矩阵 LSA 对比

🔍 对比观察：TF-IDF 矩阵通过 IDF 权重抑制了高频通用词的影响，使得 LSA 降维后语义聚类更清晰； 而 Count 矩阵保留原始频次信息，高频词可能主导降维结果。

📝 实验总结

模型核心原理优势局限性
TF-IDF + LSA词频统计 + SVD 矩阵分解无需神经网络，可解释性强无法捕获词序，维度灾难
Word2Vec神经网络分布式表示语义关系捕获好，训练高效无法处理 OOV
GloVe全局共现 + 局部上下文词类比效果优秀依赖大规模预训练语料
FastText字符 n-gram 词表示能处理 OOV 和形态变化模型体积较大
Sent2Vec (Avg)词向量均值池化简单高效丢失词序，受词向量质量制约
